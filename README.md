## 📖 Project Overview
Platform web interaktif yang dikembangkan khusus untuk **SSB Elang Mas Academy** di Banyumas Pageralang. Proyek ini dibangun dengan fokus utama pada performa tinggi, desain responsif (*mobile-first*), dan optimasi SEO untuk meningkatkan visibilitas akademi di mesin pencari.

Aplikasi ini berfungsi sebagai pusat informasi utama bagi siswa, orang tua, dan masyarakat umum mengenai jadwal, berita, dan program akademi.

## ✨ Key Features
- **SEO Optimized:** Menerapkan praktik terbaik SEO (meta tags, semantic HTML) untuk memastikan peringkat pencarian yang tinggi.
- **Mobile-First Design:** Antarmuka yang sepenuhnya responsif, memastikan pengalaman pengguna yang mulus di berbagai ukuran layar perangkat.
- **High Performance:** Waktu muat halaman yang cepat dengan optimasi aset dan teknik *rendering* modern.
- **Clean Architecture:** Menggunakan arsitektur berbasis komponen untuk skalabilitas dan kemudahan pemeliharaan kode.

## 🛠️ Tech Stack
Proyek ini dikembangkan menggunakan teknologi *frontend* modern:
- **Framework:** React / Next.js
- **Styling:** Tailwind CSS
- **Deployment & Hosting:** Vercel
- **Version Control:** Git & GitHub

## 📂 Project Structure
Pendekatan *Clean Code* diterapkan dengan struktur direktori berikut:

```text
├── src/
│   ├── components/  # Reusable UI components (Buttons, Cards, Navbar)
│   ├── pages/       # Application routes and pages
│   ├── styles/      # Global styles and Tailwind configuration
│   ├── utils/       # Helper functions and utilities
│   └── assets/      # Static files (images, icons)
```

## 🚀 Getting Started
Ikuti langkah-langkah di bawah ini untuk menjalankan proyek ini secara lokal di mesin Anda.

### Prerequisites
Pastikan Anda telah menginstal Node.js dan npm/yarn di komputer Anda.

### Installation
1. Clone repositori ini:
   ```bash
   git clone [https://github.com/Pawid-Priyanto/elangmas.git](https://github.com/Pawid-Priyanto/elangmas.git)
   ```
2. Masuk ke direktori proyek:
   ```bash
   cd elangmas
   ```
3. Instal semua dependensi:
   ```bash
   npm install
   # atau
   yarn install
   ```
4. Jalankan *development server*:
   ```bash
   npm run dev
   # atau
   yarn dev
   ```
5. Buka `http://localhost:3000` di browser Anda untuk melihat hasilnya.
